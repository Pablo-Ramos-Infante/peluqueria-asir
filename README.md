# Proyecto Peluquería ASIR

Aplicación web para gestionar reservas de una peluquería.  
Incluye:

- Sistema de reservas
- Panel de administración
- Login seguro con password_hash
- Gestión de servicios
- Listado de citas con estado
- Base de datos MySQL

## Tecnologías

- PHP 8
- MySQL / MariaDB
- Bootstrap 5
- HTML / CSS / JS

## Funcionalidades

- Reservar cita con fecha, hora y servicio
- Panel de administración con login
- Ver todas las citas ordenadas por fecha de creación
- Gestión de servicios
- Validación básica de datos
